## üéÆ Core Features

- Basic Gameplay: Side-scrolling platform jumping, crouching, pausing, with smooth operation and good feel.
- AI System: Built-in enemy AI logic, supports custom enemy behaviors, and easily realizes combat mechanisms.
- Boss Mechanism: Reserved Boss battle interface, can quickly develop exclusive Boss levels and skills.
- Item System: Supports item pickup and use, can expand various item effects (health recovery, speed boost, etc.).
- Map Parsing: Supports JSON format map files, which can be manually written or generated with the editor, enabling flexible level switching.
- Visual Editor: What you see is what you get, drag and drop to place terrain, enemies and items to quickly build game levels.
